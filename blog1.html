DRPU Barcode Label Maker Software


DRPU Barcode Label Maker Software is user friendly and easy to use, that allows user to create printable, custom and scannable barcodes labels, stickers or asset tags with many Barcode fonts such as Databar code, Databar EAN 13 code, ISBN 13, PDF417, DataMatrix, QR Code, MICR code and many more. This software creates barcode labels of different dimensions, bar width, density and height and also helps you to customize your Labels with logo, design and colors. This software facilitates you to design barcode labels in bulk amount using Sequential, Random and Constant List Generating functions. You can also import your Excel sheets which contains Barcode values to the software and generate the Barcodes.

Features of DRPU Barcode Label Maker

  • Options to linear and 2D Barcode Labels and Stickers.
  • Can form Barcode list using sequential Series, Random and Constant Value Series.
  • It has inbuilt email settings to send Barcode images at user defined email address.
  • Provides options to save Barcode images in different image file format including JPEG, TIFF, Bitmap, PNG, WMF, EMF, EXIF and GIF.
  • Allows you to copy and paste Barcode labels at specified application including MS-Word, MS-Paint, MS-Excel etc.
  • Options to print multiple Barcode labels in single paper with print preview facility.


Process to Create and Print Barcode List by importing MS-Excel File in DRPU Barcode Label Maker:


Step 1: Download and install DRPU Barcode Label Maker software on your Windows or Mac

Download Barcode Label Maker software



Step 2: Import your Excel file which contains Barcode values to software from the Batch Processing tab at right.

Step 3: You can also import Barcode Value, Barcode Header, Barcode Footer from separate Excel sheets.

Step 4: Choose between Quick Barcode Mode and Barcode Designing View Mode.

Quick Barcode Mode

Step 5(i): Check Enable Batch Processing and choose type of Barcode that you want (Linear or 2D).

Step 5(ii): Choose Barcode Value, Header and Footer. You can also edit Header and Footer values.

Step 5(iii): You can change General settings, Font settings, Color settings for the Barcodes.

Barcode Designing View Mode

Step 5(a): Add a new Barcode from Drawing Tools tab and go to its properties.

Step 5(b): Choose the type of Barcode that you want (Linear or 2D) and check Batch Processing Series.

Step 5(c): Choose Barcode Value, Header and Footer. You can also edit Header and Footer values.

Step 5(d): You can change Size, Font and Color for the Barcodes.

Step 5(e): Customize your Barcode stickers with Text, Shapes, Image, etc.

Step 6: Print your Barcodes now using print settings and your Barcodes are now ready to use.
